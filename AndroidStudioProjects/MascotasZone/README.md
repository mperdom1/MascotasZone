# MascotasZone 🐾

**MascotasZone** es una aplicación Android diseñada para fortalecer las comunidades de dueños de mascotas. Permite a vecinos y amigos organizarse para socializar a sus mascotas, ofrecer servicios de guardería (babysitting) y coordinar transporte (carpool) al veterinario.

## Características principales ✨

*   **Comunidades locales**: Crea o únete a comunidades de tu colonia o grupos de amigos.
*   **Gestión de Mascotas**: Registra a tus mascotas con su nombre, raza y edad.
*   **Solicitudes de Servicio**:
    *   **Car Pool**: Coordina viajes al veterinario con otros dueños.
    *   **Babysitting**: Ofrece o busca cuidadores de confianza dentro de tu comunidad.
    *   **Socialización**: Organiza paseos grupales para que las mascotas interactúen.
*   **Persistencia Local**: Utiliza **SQLite** para guardar toda la información de forma segura en el dispositivo.
*   **Interfaz Moderna**: Basada en Material Design 3 con navegación intuitiva por pestañas.

## Tecnologías utilizadas 🛠️

*   **Lenguaje**: Java
*   **Base de Datos**: SQLite (almacenamiento persistente)
*   **UI Components**: Material Components for Android, ConstraintLayout, RecyclerView, CardView.
*   **Arquitectura**: Fragments para la gestión de pantallas dinámicas.

## Instalación y Uso 🚀

1.  Clona este repositorio: `git clone https://github.com/TU_USUARIO/MascotasZone.git`
2.  Abre el proyecto en **Android Studio**.
3.  Sincroniza el proyecto con los archivos Gradle.
4.  Ejecuta la aplicación en un emulador o dispositivo físico (mínimo SDK 24).
